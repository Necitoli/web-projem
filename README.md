##proje açıklaması
Bu proje, kullanıcıların güreş sporunu tanıyabileceği, güreş türleri hakkında bilgi alabileceği ve efsane sporcuları inceleyebileceği bir web sitesi projesidir.
Kullanıcılar; Serbest Güreş, Greko-Romen Güreş ve Yağlı Güreş hakkında bilgi edinebilecek, önemli turnuvaları görüntüleyebilecek ve sporcuların kariyerlerini inceleyebilecektir.
Ayrıca temel güreş tekniklerini öğrenebilecek ve site üzerinden yorum yapabilecektir.

##hedef kullanıcılar
Tüm bireyler
Güreşe ilgi duyan sporcular
Antrenörler
Spor öğrencileri
Güreş severler

##temel özellikler
kullanıcı kayıt / giriş sistemi
sporcu profillerini görüntüleme
güreş türleri hakkında bilgi sayfaları
yorum yapabilme
haberler bölümü
admin paneli (sporcu ekleme / silme / güncelleme)

##kullanılan teknolojiler
HTML
CSS
JavaScript
PHP
MySQL
